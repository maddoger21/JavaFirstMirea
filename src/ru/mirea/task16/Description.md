# Практическая работа 16. Индивидуальный проект
Игра "Тамагочи".
Описание: Персонаж будет обладать некоторыми характеристиками,
которые со временем будут уменьшаться. Задача игрока не дать им упасть до 0.

У персонажа есть 3 характеристики: Здоровье, Голод, Энергия
Со временем они уменьшаются. Их состояние отображается с помощью Progress Bar

Чтобы восполнить характеристики у пользователя есть магазин с едой, 
которую он покупает за деньги. Эти деньги он приобретает, нажимая на кнопку
"Работа", которая тратит энергию у персонажа. Так что у пользователя также
есть дополнительная строка, отображающая количество денег.

Игра завершается, когда одна из характеристик доходит до нуля. Пользователю
предоставляется возможность начать игру заново.
